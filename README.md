# larou-azure-devops-status

## Project setup
```
npm install
```

## Create the configuration file and complete it
```
cd /src/
mv config.example.ts config.ts
vim config.ts
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
